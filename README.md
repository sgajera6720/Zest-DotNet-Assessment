# Zest DotNet Assessment

## 📌 Project Description
This is an ASP.NET Core Web API project developed as part of an assessment.

## 🚀 Features
- Authentication using JWT
- Student CRUD operations
- Exception handling middleware
- Repository pattern implementation

## 🛠️ Technologies Used
- ASP.NET Core Web API
- ADO.NET
- SQL Server
- JWT Authentication
- Serilog

## 📂 Project Structure
- Controllers
- Repository
- Middleware
- Models

## ▶️ How to Run
1. Clone the repository
2. Open in Visual Studio
3. Update connection string
4. Run the project
